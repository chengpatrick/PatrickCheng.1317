Tokenizer.java
Description: This is the tokenizer class file written in java

To compile this file just open it in Eclipse IDE and press run
When compiling this file, the program will ask the user to enter the file to be tokenized,the program will then read in the text in the file and tokenize them and print them out at the same time. Then the program will output a stream of numbers corresponding to the tokens in the console and to an output file.
